# 摘要

本文围绕社区志愿者服务管理场景，设计并实现了一套前后端分离的管理系统。系统采用 Spring Boot 3、Java 21、Vue 3、Vite、MySQL、JWT 和 Spring Security 等技术，完成了用户登录、角色权限控制、志愿者档案管理、志愿活动发布、报名审核、活动签到和服务时长统计等核心功能，形成了较为完整的业务闭环。后端通过统一接口、权限过滤和数据持久化机制保证业务安全，前端通过路由守卫、状态管理和组件化页面提升交互体验。测试结果表明，系统能够满足社区志愿服务管理的基本需求，具有较好的可用性和可扩展性。

**关键词：** 社区志愿者；Spring Boot；Vue 3；JWT 鉴权；MySQL

# ABSTRACT

This thesis presents the design and implementation of a community volunteer service management system based on a front-end and back-end separated architecture. The system is built with Spring Boot 3, Java 21, Vue 3, Vite, MySQL, JWT, and Spring Security. It focuses on a compact but complete workflow that includes user login, role-based access control, volunteer profile management, volunteer activity publishing, signup review, activity check-in, and service-hour statistics. On the back end, Spring Boot and Spring Security provide API development, authentication, and authorization support, while MyBatis-Plus and MySQL are used for data persistence and query processing. On the front end, Vue 3, Vite, Pinia, Vue Router, Axios, and Element Plus are used to build a component-based administrative interface with route guarding and unified request handling. The implemented system supports administrators in managing volunteer records and activities, and allows volunteers to participate in activities through a controlled workflow. The paper also summarizes the database design, interface design, and system testing process. Test results indicate that the system runs stably and can satisfy the basic requirements of community volunteer management. The project demonstrates a practical application of mainstream web development technologies and provides a feasible reference for small and medium-sized management systems.

**Key words:** community volunteer; Spring Boot; Vue 3; JWT authentication; MySQL
